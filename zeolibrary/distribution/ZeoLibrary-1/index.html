ZeoLibrary

A library by evsc for the Processing programming environment.
Last update, 08/22/2012.

Load and analyze Zeo Sleep Manager's CSV export file.
Compute average sleep duration, REM, light, deep sleep values. Upcoming addition: ZeoStream, to stream live-data from the Zeo Bedside Sleep Manager.
Export your Zeo Sleep Manager data from their website, and import the .csv file into Processing. The library dissects the data, cleans out unusable nights, let's you access your sleep data easily and computes average values of your various sleep data values (total sleep, REM, light sleep, deep sleep, sleep onset time, awakenings, ...)

Download

Download ZeoLibrary version 0.1.1 (1) in .zip format.

Installation

Unzip and put the extracted ZeoLibrary folder into the libraries folder of your Processing sketches. Reference and examples are included in the ZeoLibrary folder.
